# Lord of the Mysteries Fan Project

A comprehensive, interactive fan website dedicated to exploring the lore, characters, and world-building of the novel **"Lord of the Mysteries"** by Cuttlefish That Loves Diving.

---

## 📖 Project Overview

This is an unofficial fan project created for educational and entertainment purposes. The site provides:

- **Novel Information** – Overview of the series, awards, and recognition
- **Character Profiles** – Detailed profiles of main characters (MC/protagonist)
- **Pathway System** – Breakdown of the unique Beyonder pathways
- **Church & Organizations** – Lore on the various factions and groups
- **Gallery** – Visual collection of fan art and relevant imagery
- **Quotes & Songs** – Memorable quotes and fan-created music
- **Volume Summaries** – Chapter-by-chapter breakdowns of all 8 volumes

---

## 🛠️ Tech Stack

- **HTML5** – Semantic markup and structure
- **CSS3** – Modern styling with animations and gradients
- **JavaScript (Vanilla)** – Interactive components (tabs, accordions, smooth scrolling)
- **Responsive Design** – Mobile-friendly layout

## 🚀 How to Use

1. **Local Development:**
   ```bash
   # Clone or download the project
   cd LOTM
   
   # Open in a browser
   open LOTMhome.html
   ```

2. **Navigation:**
   - Use the top navigation bar to explore different sections
   - Click the back button (↑) in the top-right to return to the main portfolio
   - Use the "Scroll to Top" button (↑) to quickly jump to the top of the page

3. **Responsive Design:**
   - The site is fully responsive and works on desktop, tablet, and mobile devices
