# Pudding Paradise

A responsive dessert-themed landing site with a menu page, product showcase, and review page.

## Overview

Pudding Paradise is a static HTML/CSS project designed as a charming dessert storefront experience. It includes multiple styled pages, rich pastel visuals, and a playful user interface built for mobile-first presentation.

## Pages

- `menu.html` — dessert menu and featured product sections
- `puddinghome.html` — home-style landing page with illustrations and shop highlights
- `review.html` — customer review and testimonial page

## Built With

- HTML
- CSS
- Google Fonts (`Quicksand`)
- Font Awesome icons

## Features

- Responsive mobile-first layout
- Pastel dessert-themed color palette
- Animated buttons and cards
- Accessible typography and spacing
- Multi-page navigation for a simple storefront experience

## Usage

1. Open any of the HTML files directly in your browser.
2. No build step or local server is required.
