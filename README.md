# My Projects

This workspace contains several web app and UI projects built with HTML, CSS, JavaScript, and React. Each project is stored in its own folder with files and assets specific to that project.

## Projects

### Origami AR

- Interactive web experience using webcam hand tracking, Three.js, React, and shader effects.
- Built with Vite, TypeScript, `@react-three/fiber`, and MediaPipe.
- Folder: `Origami AR`

### Pudding Paradise

- Dessert-themed static storefront site with menu, home, and review pages.
- Built with responsive HTML and CSS.
- Folder: `Pudding Paradise`

### Scratchpad

- Note-taking web app with theme selection, localStorage persistence, and sectioned notes.
- Built with HTML, CSS, and JavaScript.
- Folder: `Scratchpad`

### TheAkashic

- Story collection and progress tracker web app.
- Contains an existing `README.md` in the project folder.
- Folder: `TheAkashic`

## Root Files

- `index.html` — My landing page.
- `portofolio.html` — My personal portfolio page.
- `style.css` — styles for the root pages.

## Notes

- If you want to run `Origami AR`, navigate to `Origami AR` and use `npm install` followed by `npm run dev`.
- `Pudding Paradise` and `Scratchpad` can be viewed by opening their HTML files in a browser.
- `TheAkashic` already contains its own project-level documentation.
