About

How I Work

I'm a vocational school student focused on front-end web development. I build multi-page sites from scratch using semantic HTML, CSS layout systems (flexbox and grid), and vanilla JavaScript for interactivity and DOM manipulation.

Every project in this portfolio is hand-coded — no templates, no page builders, no frameworks. I'm currently working toward writing cleaner, more maintainable code, deepening my understanding of responsive design patterns, and practicing version control with Git.
